
# C# 复制列表

本文：如何复制一个列表

<!--more-->



<div id="toc"></div>

最简单的方法是 foreach


```csharp
   foreach(var temp in a)
   {
       b.Add(temp);

   }
```

有没一个简单的方法？


```csharp
            using System.Linq;

            var a = new List<Fex>()
            {
                new Fex() {F = true,},
                new Fex() {F = true,},
                new Fex() {F = false,},

            };
            List<Fex> b = a.ToList();

            b.RemoveAt(0);

            Console.WriteLine(a.Count);

            b.Add(new Fex());
            b.Add(new Fex());

            Console.WriteLine(a.Count);
```

            
`List<Fex> b = a.ToList();` 可以把列表a到列表b，对b进行删除、添加，不会对a造成元素改变。
