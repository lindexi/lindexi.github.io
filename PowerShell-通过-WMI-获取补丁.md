
# PowerShell 通过 WMI 获取补丁

本文告诉大家如何通过 WMI 获取补丁

<!--more-->


<!-- csdn -->

<!-- 标签：PowerShell,WMI -->

通过 Win32_QuickFixEngineering 可以获取系统启动的服务

```csharp
Get-WmiObject Win32_QuickFixEngineering
```
