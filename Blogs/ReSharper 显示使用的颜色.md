此功能名叫 “Highlight color usages” 可以对代码里面的颜色进行颜色标识，比如在代码提示或在编写完成里显示代码的颜色对应的颜色方块。可在 ReShaprer 的选项的 Code Inspection 的 Settings 里面，通过勾选或去掉勾选 Highlight color usages 来决定功能的开关
