# WPF 提高性能

本文告诉大家如何使用 dot net core 的方法提高 WPF 的性能。因为微软开源dot net core 所有很多大神看到垃圾微软的代码，把微软的代码性能提高，微软在 4.7.1 发布就有这些性能的代码。但是现在使用的 WPF 使用的框架是 .net 4.5 无法升级，所以只好使用复制源代码的方式提高 WPF 代码的性能。
